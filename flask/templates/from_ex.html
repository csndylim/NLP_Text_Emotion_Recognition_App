😍 Text Emotion Recognition 😨

            By Tan Jun Xian, Candy Salome Lim, Afiqah Aris

            Built using RoBERTa model with Crowdflower dataset.
            Trained for 12 hours, 15 minutes and 4 seconds.

            Model parameters:
            The model was fine tuned using 3-Fold Cross Validation, 
            with Categorical Entropy Loss as this is a multi-class classification.
            The AdamW optimizer is also used with a Learning Rate of 1e-5, 
            in which the ReduceLROnPlateau function was implemented, with a patience of 3. 
            The Max Length parameter has also been set to 256, with Batch Size of 8, 
            this is due to the limitations of the available GPU.

😤 How to run? 😫

            1. Enter the text in English/Chinese/French/German/Italian/Japanese/Korean/Malay/Spanish in the input section.
            2. The model will predict the top 3 most probable emotions in English.
            3. Results will be shown below.
